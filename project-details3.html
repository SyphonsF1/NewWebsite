01 Introduction

The 2021 states car that we had created had a lot of things in mind when we underwent the design process of it. The 2021 Syphons F1 in schools car, has undergone a lot of design changes and new innovations. These innovations included: The void running down the center of the car - This void that is running down the center of the car, this design feature has been placed here of one of two reasons. The void has been designed to reduce the air resistance on the car, the second reason for this design feature is that it allows the air to flow with more ease. Some con’ of this design feature is the two tetherline guides that are in the void under the car. Due to these tetherline guids being here they create a lot of drag, but in virtual testing it has been decided that is this void was not here the drag created would be a lot higher. The Nose Cone - This design feature is one of the most important parts of the car, this is because if this part was absent or it was a different shape, it would create a lot more drag that would make the car very slow. The reason that this design feature is so important is because it allows any air the did not go down the center void of the car can flow over the car to create as little drag as possible.
